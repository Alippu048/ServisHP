# Kasir Servis HP

Aplikasi desktop berbasis Java Swing untuk mengelola transaksi servis handphone.

## Teknologi
- Java
- Java Swing
- MySQL
- NetBeans IDE

## Fitur
- Login Admin
- Manajemen Data Customer
- Transaksi Servis
- Perhitungan Total Otomatis
- Diskon
- Penyimpanan Data MySQL

## Cara Menjalankan
1. Import database `servishp.sql`
2. Buka project di NetBeans
3. Jalankan Main.java
